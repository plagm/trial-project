# Contributing to InvoiceLoop

Thank you for your interest in contributing to InvoiceLoop! Here are some guidelines to help you get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/invoiceloop.git`
3. Install dependencies: `npm install` in both `frontend` and `backend` directories.
4. Set up environment variables based on `.env.example`.
5. Start development servers:
   - Frontend: `npm run dev` in the `frontend` folder
   - Backend: `npm run dev` in the `backend` folder

## Pull Requests

- Keep PRs focused on a single issue.
- Write clear, descriptive commit messages.
- Ensure your code follows the existing style guidelines.
- Add tests if applicable.

## Issues

If you find a bug or have a feature request, please open an issue in the issue tracker. Provide as much detail as possible, including steps to reproduce bugs.
